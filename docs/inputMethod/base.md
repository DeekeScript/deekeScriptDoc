# 输入法设置

> 特别说明：APP打包后，对应的输入法名称会变成“XXX输入法”，其中“XXX”为APP名称

## 开启步骤

* 打开手机的“设置”页面
* 搜索“输入法”
* 点击“默认”
* 最终选择“DeekeScript输入法”为默认输入法

> 注意：必须将DeekeScript输入法选择为默认
